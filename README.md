## 项目简介

⭐ 为在同类型工具中推荐尝试程度，程度高说明在同类工具中相对能力更强（有不同于同类工具的功能）

### 媒体工具
* [EasyRip](https://github.com/op200/EasyRip) ⭐⭐⭐  
  功能齐全且用法简单的压制工具，新手也可以一键使用，适合各种制作组作为入门压制工具
* [Windows Font Manager](https://github.com/op200/windows_font_manager) ⭐⭐⭐  
  展示并可以简单操作 Windows 中的字体
* [Video Match](https://github.com/op200/video_match) ⭐  
  生成两个视频的画面映射，可以用于不同视频版本之间的时轴同步
* [Video Quality Compare](https://github.com/op200/synchrsub)  
  简陋的视频质量比较器

### 字幕工具
* [Tag Replace](https://github.com/op200/Tag-Replace_for_Aegisub) ⭐⭐⭐  
  可以代替 karaoke template 的 Aegisub 插件，所有操作无副作用，用法偏 Lua 底层，内置大量工具
* [SSOCR](https://github.com/op200/Simple_Subtitle_OCR) ⭐  
  简陋的 OCR 硬字幕的 GUI
* [synchsub](https://github.com/op200/synchrsub)  
  未开发完的 Web ASS 编辑器，浏览器打开在线使用链接即可使用

### BT 工具
* [oidbt_torrent](https://github.com/OIDBT/oidbt_torrent)  
  Python 的 Torrent 对象操作库
* [torrent-parser](https://github.com/op200/torrent-parser) ⭐  
  Web 的 Torrent 解析器，浏览器打开在线使用链接即可使用
* [lt_make_torrent](https://github.com/op200/lt_make_torrent) ⭐⭐  
  基于 Libtorrent 的 CLI 制种工具，用于替代 mktorrent 等工具
* [oidbt-bangumi-app](https://github.com/OIDBT/oidbt-bangumi-app) ⭐⭐  
  一个 Bangumi 插件，基于 IPFS 去中心化磁链获取，从 Bangumi 之间打开磁链

### 其他小工具
* [my_Gadgets](https://github.com/op200/my_Gadgets)  
  存了各种随手写的小工具的仓库
* [manbo-voice](https://github.com/op200/manbo-voice) ⭐  
  Web 的 曼波 语音生成器，生成功能源于第三方 API  
  该项目功能为自动拆分长文本以批量生成，且能让生成的语音断句自然  
  浏览器打开在线使用链接即可使用
* [ADS](https://github.com/op200/ADS)  
  Web 量表，网上找不到免费用的在线量表，就随手写了个  
  浏览器打开在线使用链接即可使用
* [bangumi-new-ani-remind](https://github.com/op200/bangumi-new-ani-remind)  
  一个 Bangumi 插件
* [anime-table-creater](https://github.com/op200/bangumi-new-ani-remind)  
  当时 bingo 等填格子游戏很流行，就随手写了个，能自动填格子
